# PostFinance Checkout Woocommerce
This repository contains the PostFinance Checkout WooCommerce plugin that enables WooCommerce to process payments with [PostFinance Checkout](https://postfinance.ch/en/business/products/e-commerce/postfinance-checkout-all-in-one.html).

##### To use this extension, a [PostFinance Checkout](https://postfinance.ch/en/business/products/e-commerce/postfinance-checkout-all-in-one.html) account is required.

## Requirements

* [Wordpress](https://wordpress.org/) 4.4 or later
* [Woocommerce](https://woocommerce.com/) 3.0 or later
* [PHP](http://php.net/) 5.6 or later
