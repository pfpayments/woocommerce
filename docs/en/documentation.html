1Prerequisites

If you don’t already have one, create a PostFinance Checkout account and set up a space by following the setup assistant.

2Installation

  1. Install it directly from your Wordpress Installation under Plugins > Add New search for PostFinance Checkout.

  2. The plugin will now be visible under Plugins. You can install the plugin directly from here. After the plugin is installed, you can open the configuration.

3Configuration

  1. The plugin’s configuration can be found under WooCommerce > Settings > PostFinance Checkout or click the link on the plugin site.

  2. Enter the credentials from before which means the PostFinance Checkout application user’s access information and the space id. The configuration will be done automatically.

4Payment Method Configuration

Most of the payment method configuration is done in PostFinance Checkout. After following the steps of the basic configuration, the PostFinance Checkout payment method configuration will be synchronized automatically into the WooCommerce shop.

The synchronized methods are listed under WooCommerce > Settings > Payment.

5Refunds

A refund at PostFinance Checkout can be in a pending state. There are different reasons for this e.g: The processor is not reachable at the time or the processor executes refunds as a batch at a specific time of the day. The WooCommerce Refund object will be in the state 'pending' as well. After the refund is in a final state, it will be automtically updated in your shop. If you have automated process for refunds, ensure they are only executed if a refund is in the 'completed' state.

6FAQ

6.1How can I make the payment methods appear in the checkout?

Make sure that you followed the Configuration section by stating your PostFinance Checkout space ID and application user’s access information in the settings page. By saving the configuration form the synchronization of the payment methods and the set up of the webhooks are initiated. Ensure the payment method are visible in the WooCommerce Settings > Payment section and the methods are active.

If this does not solve the problem, make sure no other plugin is hiding the payment methods.

6.2How can I translate the plugin?

  1. Install the Wordpress Loco Translate plugin.

  2. Go to Loco Translate > Plugins.

  3. Click on WooCommere PostFinance Checkout.

    1. Click on the existing language and adapt the translations.

    2. Cick New Langugage and translate the strings.
